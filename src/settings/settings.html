Obsidian
Setup instructions
  1. Install the Local REST API plugin in Obsidian
  2. Enable it and copy the API key from plugin settings
  3. Install the Advanced URI plugin (for stable deep links)
  4. Paste the API key above and enter your vault name
Browser
Setup instructions
  1. Click Open Extension Folder above
  2. In Chrome: go to chrome://extensions, enable Developer mode, click "Load unpacked" and select the folder
  3. Copy the extension ID and paste it above
  4. Click Register Bridge to set up the native messaging host
  5. Without the extension, browser URLs are detected via window title (slower)
