Obsidian
Setup instructions
1. Local REST API (by Adam Coddington)
  1. Open Obsidian → Settings (gear icon, bottom-left) → Community Plugins → Browse
  2. Search "Local REST API" → Install → Enable
  3. Go back to Settings → scroll to "Local REST API" under Community Plugins
  4. The API Key is shown at the top of the plugin's settings panel — click the copy button and paste it below
  5. Leave all other settings at their defaults (HTTPS on port 27124)
2. Advanced URI (by Vinzent03)
  1. Same steps — search "Advanced URI" in Community Plugins → Install → Enable
  2. In the plugin settings, turn on "Use UID instead of file paths"
  3. This makes note links survive file renames. All other settings can stay at their defaults.

Once both plugins are enabled and Obsidian is running, click Test Connection below to verify.

How it works

            
Vikunja Quick Entry automatically reads the URL from the active browser tab when Quick Entry opens. This works with Chrome, Firefox, Edge, Brave, Opera, and Vivaldi — no extension required.

            
Automatic detection

            
No setup needed. URLs are detected automatically.
